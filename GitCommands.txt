Installtion:

1. Open browser: https://git-scm.com/downloads
2. Download the git 
3. Open installed Git 
4. Make setup the the git 

Commands to check the version of the Git
> git --version
> git config
> git config --global --list

Setup for --global --list
> git config --global user.name "name"
> git config --global user.email "email_id"
